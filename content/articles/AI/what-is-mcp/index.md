## MCP って何？

Model Context Protocol (MCP) は、AI アプリケーションを外部システムに接続するためのオープン標準プロトコルです。
MCP を使用することで、Claude や ChatGPT などの AI アプリケーションからデータソース(ローカルファイル、データベースなど)やツール(検索エンジン、計算機など)、ワークフローに接続し、重要な情報にアクセスしたり、タスクを実行できるようになります。

What is the Model Context Protocol (MCP)? - Model Context Protocol
https://modelcontextprotocol.io/docs/getting-started/intro
