# Requirements: コードブロックのコピーボタン改善

## 概要
ブログ記事のコードブロックに表示されるコピーボタンのUX改善

## 要求内容

### 1. ホバー時のみ表示
- コピーボタンは普段は非表示にする
- コードブロックにマウスカーソルを合わせた時のみ表示する

### 2. アイコン化
- 「Copy」テキストではなくアイコン（クリップボードマーク）を使用する
- コピー完了時はチェックマークアイコンを表示する

### 3. コピー完了状態の表示
- コピー完了マークは1.2秒間表示する
- コピー完了中はマウスカーソルの位置に関係なく表示を維持する
- 完了後にマウスが離れている場合は、コピーアイコンを表示せずに非表示にする（チラつき防止）

## 受け入れ条件
- [x] コードブロックにホバーするとコピーボタンが表示される
- [x] ホバーを外すとコピーボタンが非表示になる
- [x] コピーボタンはクリップボードアイコンで表示される
- [x] コピー成功時はチェックマークアイコンに変わる
- [x] コピー完了状態は1.2秒間維持される
- [x] コピー完了後、マウスが離れていればアイコンがチラつかずに消える
