# 要求内容

## 概要
記事内の `https://` で始まるURLを自動的にクリック可能なリンクに変換する機能を追加する。

## 背景
- Markdown記法の `[テキスト](URL)` 形式ではなく、URLをそのまま書くだけでリンクにしたい
- Zolaはネイティブで自動リンク機能をサポートしていない

## ユーザーストーリー
- 記事作成者として、URLをそのまま書くだけでリンクになってほしい
- 読者として、URLをクリックして参照先に移動したい

## 受け入れ条件
- [x] `https://` または `http://` で始まるURLが自動的にリンクになる
- [x] コードブロック内のURLはリンク化されない
- [x] 既存のリンク内のURLは二重処理されない
- [x] リンクは新しいタブで開く

## 制約事項
- ZolaのMarkdownパーサー（pulldown-cmark）は自動リンクをサポートしていないため、JavaScriptで実装
