# 要求内容

## 概要
Markdownのコードブロックおよびインラインコードの見た目を改善する。

## 変更要求

### 1. シンタックスハイライトテーマの変更
- 現状: `material-theme-ocean`（背景が黒すぎる）
- 要望: より優しい色合いのテーマに変更

### 2. インラインコードのスタイル改善
- 背景色と文字色を目立たせる
- ボーダーを少し狭くする
- 角丸を追加する

### 3. コードブロックの角丸
- コードブロックの角を少し丸くする

### 4. コピーボタンの改善
- 角丸を追加
- 横スクロール時にボタンが一緒にスクロールしないよう固定
- フェードイン/アウトのアニメーション追加
- コピー完了時のチェックマークサイズ調整
- コピー完了時のボーダーを内側に表示

## 受け入れ条件
- [x] シンタックスハイライトが見やすくなっている
- [x] インラインコードが目立ち、読みやすい
- [x] コードブロックの角が適度に丸い
- [x] コピーボタンが横スクロールしても固定位置に表示される
- [x] コピーボタンの表示/非表示がスムーズにアニメーションする
- [x] コピー完了時のボタンサイズが変わらない
