# プロダクト要求定義書

## 概要

| 項目 | 内容 |
|------|------|
| プロダクト名 | etak64n's blog |
| URL | https://blog.etak64n.dev/ |
| リポジトリ | https://github.com/etak64n/etak64n-blog |

## プロダクトビジョンと目的

技術記事を中心としたパーソナルブログサイト。
クラウドサービス、セキュリティ、開発ツールなどの技術情報を発信し、知識の整理と共有を目的とする。

## ターゲットユーザー

- 主な読者: 技術に興味のあるエンジニア・開発者
- 著者: サイトオーナー（etak64n）

## 主要な機能一覧

### コンテンツ機能

| 機能 | 説明 |
|------|------|
| 記事表示 | Markdown で書かれた記事をHTML として表示 |
| カテゴリ分類 | 記事をトピック別カテゴリで整理（AI, AWS, Cloudflare 等） |
| タグ付け | 記事に複数タグを付与して横断的な分類を実現 |
| 目次生成 | 記事内の見出しから自動で目次を生成 |
| 関連記事表示 | タグの重複度に基づいて関連記事を自動表示 |
| ヒーロー画像 | 記事ごとにヒーロー画像を設定可能（未設定時はプレースホルダー） |

### レイアウト・UI 機能

| 機能 | 説明 |
|------|------|
| グリッドカード表示 | トップページ・一覧ページでサムネイル付きカード形式で記事を表示 |
| スティッキー目次 | 記事ページ左側に固定表示される目次 |
| レスポンシブデザイン | PC・タブレット・スマートフォンに対応 |
| タクソノミーページ | タグ・カテゴリごとの記事一覧ページ |

### 運用機能

| 機能 | 説明 |
|------|------|
| 下書き機能 | `draft = true` で公開前の記事を管理 |
| サイトマップ生成 | SEO 向けに sitemap.xml を自動生成 |

## 非機能要件

### パフォーマンス

- 静的サイトとしてビルドし、高速な表示を実現
- Cloudflare のエッジネットワークを活用した低レイテンシ配信

### 保守性

- Markdown ベースの記事管理で執筆が容易
- カテゴリフォルダによる記事の整理

### 可用性

- Cloudflare Pages によるグローバル配信
- GitHub Actions による自動デプロイ
